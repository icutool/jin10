【观点】南华期货：焦煤现货难以维系

【盘面信息】焦煤期货大跌，跌幅近5%。

【信息整理】1. 3月31日讯，按照焦煤长协煤钢联动方案数据测算，2026年3月焦煤长协煤钢联动浮动值比2026年2月下跌24元/吨，跌幅1.6%。（中国钢铁（850106）工业协会）

2. 2026年1-2月份，国内钢铁（850106）行业利润累计值为-24.7亿元，同比增幅-59.35%。

3. 根据彭博，伊朗冲突引发的中东能源（850101）供应中断，尤其是液化天然气（885430）（LNG）运输受阻，导致全球天然气（885430）价格大幅上涨，欧洲与亚洲能源（850101）成本迅速攀升。在供应不稳定与价格飙升的双重压力下，多国被迫重新转向煤炭（850105）发电，以保障电力和工业运行。煤炭（850105）因供应稳定、成本较低，重新成为“兜底能源（850101）”，推动其需求与价格同步回升。这一趋势使原本推进的能源（850101）转型出现阶段性倒退，欧洲和亚洲部分国家延长煤电使用周期（883436）甚至重启煤电项目。整体来看，全球能源（850101）体系正从“低碳优先”转向“安全优先”，短期内煤炭（850105）受益，但代价是碳排放上升与气候目标受挫。

4. 在伊朗冲突引发中东能源（850101）紧张的背景下，日本作为高度依赖进口能源（850101）的国家，正面临天然气（885430）价格飙升与供应不确定性的双重压力。尽管LNG供应尚未完全中断，但价格已大幅上涨，使天然气（885430）发电成本显著高于煤电。在此情况下，日本被迫增加煤炭（850105）使用，以保障电力系统稳定运行，推动澳大利亚等高品质煤需求上升，并在亚洲市场形成“高价抢煤”的局面，挤压其他中小进口国。短期内，煤炭（850105）重新成为日本能源（850101）结构中的关键支撑，而核电（885571）重启进展缓慢，难以快速替代。整体来看，日本的能源（850101）策略正在从“低碳优先”转向“成本与安全优先”，并成为推动全球煤价上涨的重要力量。

【南华观点】焦煤主力合约大跌逾5%，并未显著事件驱动，主要原因在于近期市场情绪走弱，特别是现货价格松动，导致期货高估值难以维系，价格有下修需要。目前焦煤整体供给充裕，库存边际累库，现货端不缺，且有降价行为。此外能源（850101）逻辑有所反复，因此期货需要回落消化其估值。目前下跌后焦煤基差有所修复，风险释放后整体下方空间有限。
